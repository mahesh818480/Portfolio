Apartment Booking Application

Developed a comprehensive apartment booking application in Angular that allows users to browse apartments, select rooms, and manage bookings. Users can view room details such as square footage, number of bathrooms, and additional amenities, and check room availability in real-time. The application also tracks booking status, including confirmed and pending bookings, and calculates the total amount payable based on selected rooms and features.

This project emphasizes user-centric design, interactivity, and real-world functionality, providing a seamless booking experience. Features like floor selection, room configuration, real-time validations, and visual feedback make the application both engaging and reliable.

Key Highlights

  • Built a dynamic room and floor selection system with detailed specifications for each apartment
  • Implemented real-time availability tracking, showing which rooms are booked and which are pending
  • Added booking management features, including confirmation status and pending requests
  • Developed interactive forms with validations and conditional fields for accurate bookings
  • Calculated room prices dynamically based on selected rooms and amenities
  • Provided responsive and intuitive UI for desktop and mobile users
  • Applied unit testing for forms, components, and booking logic to ensure reliability
  • Designed the application for scalability, allowing future integration with backend APIs and payment gateways
